1. ISI PACKAGE (.ZIP)
------------------------------------------------------------
- index.html      : Struktur utama website.
- assets/         : Folder untuk gambar dan ikon (jika ada).
- README.txt      : Panduan yang sedang Anda baca ini.

------------------------------------------------------------
2. CARA INSTALASI
------------------------------------------------------------
1. Extract file .zip ke folder kerja Anda.
2. Buka 'index.html' menggunakan browser (Chrome/Edge/Brave).
3. Untuk editing, gunakan VS Code atau Sublime Text.

------------------------------------------------------------
3. FITUR EKSKLUSIF
------------------------------------------------------------
- Animate On Scroll (AOS): Elemen akan muncul secara dinamis saat di-scroll.
- Custom Cursor: Interaksi cursor minimalis untuk pengalaman user yang unik.
- Glassmorphism UI: Efek kartu transparan modern.
- Tailwind CSS Support: Mudah dikustomisasi tanpa perlu nulis CSS ribet.

------------------------------------------------------------
4. CARA MODIFIKASI (KUSTOMISASI)
------------------------------------------------------------
- GANTI TEKS: Cari tag <h1> atau <p> di index.html dan ganti sesuai pesan Anda.
- GANTI WARNA: Ubah kode HEX pada bagian 'background-color' di tag <style>.
- MATIKAN CURSOR: Jika ingin kursor standar, hapus bagian <div id="custom-cursor"> 
  dan hapus script kursor di bagian bawah file.
